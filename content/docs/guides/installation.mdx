## Prerequisites

Before installing Glide MCP, make sure you have the following:

- Node.js 18+ installed
- Git configured on your system

## Installation Steps

### Step 1: Download Glide MCP

```bash
npm install kite
```

### Step 2: Configure

Follow the setup wizard to configure Glide MCP for your project.

### Step 3: Verify Installation

```bash
kite --version
```
